Este programa en Java permite ordenar un arreglo de tamaño par desplazando los elementos de la primera mitad a la izquierda y los de la segunda mitad a la derecha,
según un valor T que el usuario ingresa. El programa solicita al usuario que ingrese los valores del arreglo y el valor de T, luego desplaza los elementos y muestra el resultado final. El rendimiento del programa depende de T y del tamaño del arreglo, con una complejidad temporal de O(n log n), donde N es el tamaño del arreglo y T el número de desplazamientos. A medida que T aumenta, el tiempo de ejecución crece linealmente.
